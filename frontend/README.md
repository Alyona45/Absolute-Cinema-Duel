# Фронтенд — минимальный клиент Flappy

В этой папке находится минимальный статический клиент для ручного тестирования WebSocket‑эндпоинта игры Flappy.

Использование:

1. Поднимите статический сервер (или откройте файл в браузере):

```bash
# из корня репозитория
python -m http.server --directory frontend 8001
# затем откройте http://localhost:8001/flappy.html
```

2. Укажите хост/порт/ID сессии и нажмите `Connect`.
3. Нажмите `Jump` или пробел, чтобы отправить событие прыжка на сервер.

Клиент ожидает JSON‑сообщения от сервера согласно схемам в `backend/schemas/game_flappy.py`.
